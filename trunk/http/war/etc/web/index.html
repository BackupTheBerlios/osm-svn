Welcome to OSM Web on Metro 1.0 and Jetty 6

This osm web server component is based on a beta edition of Jetty 6.

This is a test context resolved from a WAR file via DPML Transit that serves:

  • static content ( tiny, small, medium, large, compressed)
  • a Request Dump Servlet
  • a Session Dump Servlet
  • a Request Dump JSP
  • a Hello World Servlet

A static resource handler context provides hooks into the local documentation.

  • a Installed DPML site docs
  • a DPML Data Directory

Technical Note

Improvements within Jetty 6.0.x include:

  • Totally rearchitected and rebuild relative to JDK1.4
  • Improved "dependancy injection" and "inversion of control" design of components
  • Improved "interceptor" design of handlers
  • Smart split buffer design allows large buffers to only be allocated to active connections. The resulting memory savings allow very large buffers to be used, which increases the chance of efficient asynchronous flushing and of avoiding chunking.
  • NIO Buffering so that efficient direct buffers and memory mapped files can be used.
  • NIO non-blocking scheduling so that threads are not allocated per connection.
  • NIO gather writes, so that for example a HTTP header and a memory mapped file may be sent as sent is a single operation.
  • Continuations for efficient AJAX polling
