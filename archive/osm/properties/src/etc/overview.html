The net.osm.properties package provides support for the management of property sets attributed to remotely accessible objects.

Components
{@link net.osm.properties.PropertySetDefServantManager} ServantManager implementation that provides support for the management of remotely accessible properties. The coponent exposes the service interface {@link net.osm.properties.PropertiesService} through which existing properties can be accessed and new properties created.
