The net.osm.sps package provides support for the establishment of Structured Push Supplier (SPS) servant that handles the queuing and dispatching of structred events to a target consumer.

Components
{@link net.osm.sps.StructuredPushSupplierServantManager} ServantManager implementation that provides support for the queueing and dispatch of events to a remote consumer.
