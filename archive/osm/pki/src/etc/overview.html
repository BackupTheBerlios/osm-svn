The net.osm.pki package provides support for the establishment of PKI infrustructure supporting certificate creation, extension, revocation and certificate retrival.

Components
{@link net.osm.pki.repository.RepositoryServer} PKIRepository implementation that provides support for the storage and retrival of digital certificates.
{@link net.osm.pki.authority.RegistrationAuthorityServer} RegistrationAuthority implementation that provides support for certificate issuance.
