The net.osm.pki.repository package impements the OMG PKI Repository specifications enabling Principal discovery, pricipal publiscation, modification and deletion, and access to repository provider, implementation and repository policy information.

The server provides support for the following functions:

  • access to repository provider information
    • product version
    • product vendor
    • spec version supported
    • product description
    • supported certificate types and encoding
    • supported CRL types and encoding
    • supported cross certificates
  • publish a new PKI Principal to the repository
  • locate a PKI Principal by name
  • update an existing Principal entry
  • delete an existing Principal

The platform uses the OMG PSS (Persistent State Service) as the storage engine behind the server. This enables private management of the mapping of state to persistence independently of the server interfaces. The implementation provides a destinct seperation of the functional concerns of a Principal directroy services as oposed to repository maintenance (Principal registration, updating and deletion).
