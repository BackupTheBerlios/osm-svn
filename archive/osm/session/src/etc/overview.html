Package Description

The net.osm.session package contains abstract servides supporting the establishment and management of people, places, things and business processes.

Components
{@link net.osm.session.HomeProvider} GatewayProvider is a component implementation that handes the establishment of services supporting Task, Workspace, User and Processor services. The implementation supports the resolution of user references using the underlying security principal and provides full support for client side adapters (enabling simplified application develoipment). Initial services available to clients are accesible directly via the gateway lookupoperation, including access to available processor logic extensions registered under an ApplianceChooser.
